# Stopwatch
**Features**

In this app, the user will be able to do following:

- start the timer
- stop the timer
- reset the timer


**Approach**

This app build using `tkinter 3.9.14` and `datetime`.
- Import the datetime and tkinter modules.
- `strftime()` gives the time as `Hours:Minutes:Seconds` format
